Install Python:

1. Install the latest version of Python by clicking here

2. Make sure the version is 3.6 or higher

3. When installing, make sure you add Python to your PATH (shown in the picture on the left)

Setup alles:

4. Download the alles folder by clicking here

5. Once done installing Python, click "setup." in the alles folder

6. Make sure to install the Google Chrome browser when prompted to do so

NOTE:
For easier access, right click "alles." and click "Pin to taskbar". Dragging the application to your desktop will not work.
